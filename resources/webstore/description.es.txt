Esta extensión te permite descargar tus palabras del diccionario LinguaLeo como un fichero CSV. El fichero está listo para ser importado a Anki.

Una vez añadida, deberás ver el botón de "Exportar" en todas las páginas del diccionario.
Funciona para todos los conjuntos de palabras y soporta todos los filtros integrados de Lingualeo.

Para cada palabra descarga:
- traducción
- enlace a imagen
- transcripción
- ejemplo de uso (contexto)
- enlace a archivo mp3 con pronunciación
- conjunto de palabras en las que la palabra está incluida
- ejemplo de uso (contexto) para cartas cloze

Código fuente: https://github.com/troggy/anki-leo
