Данное расширение позволяет скачать ваши слова из словаря LinguaLeo, после чего их можно использовать по своему усмотрению. Например, добавить в программу интервальных повторений Anki.

Инструкция по импорту слов в Anki: http://troggy.github.io/anki-leo/

После установки расширения на страницах словаря и словарных наборов появится кнопка «Скачать».

Можно скачивать как и все слова из словаря сразу, либо только неизученные или выбранные. Так же можно скачивать только слова из определенного набора. Учитываются фильтры Лингуалео (слова/предложения/фразы и новые/на изучении/изученные). Все фильтры комбинируются, то есть, например, можно скачать только неизученные фразы из определенного набора и т.п.

Для каждого слова скачиваются:
- перевод
- ссылка на картинку
- транскрипция
- пример использования (контекст)
- ссылка на mp3-файл с произношением
- наборы слов, в которые слово входит
- пример использования (контекст) для карточек cloze

Исходный код: https://github.com/troggy/anki-leo
